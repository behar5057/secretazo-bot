# 🤫 SecretAzo - Anonymous Venting Bot

A Telegram bot that allows users to share secrets anonymously. Messages are reviewed by admin and published to a channel.

## ✨ Features
- Anonymous messaging
- Admin review system
- Media support (photos, videos, voice)
- Automatic channel posting
- User ban system
- Privacy focused

## 🚀 Quick Start

1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Create `.env` file with your tokens
4. Run: `python src/bot.py`

## 📦 Deploy on Render

1. Fork this repository
2. Connect to Render
3. Add environment variables
4. Deploy!

## 🔧 Environment Variables
- `BOT_TOKEN`: Your bot token from @BotFather
- `CHANNEL_USERNAME`: Channel username (e.g., @secretazo_channel)
- `ADMIN_USER_ID`: Your Telegram user ID
